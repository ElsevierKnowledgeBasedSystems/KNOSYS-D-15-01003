Welcome to the XJAF home!

XJAF is a multi-agent framework based on the Java Platform, Enterprise Edition (Java EE), one of the leading industry standards for developing scalable, secure, and reliable software. Java EE offers a wide variety of technologies that can be used to implement many functionalities of a multi-agent platform. The main idea behind the development of XJAF is, therefore, to avoid "reinventing the wheel," and utilize the technical solutions readily-available in Java EE.

For example, XJAF agents are mapped to Enterprise JavaBeans (EJBs), server-side Java EE components that encompass the business logic of an application. By utilizing the EJB technology and running on top of a computer cluster, XJAF can easily offer the following advanced functionalities: load-balancing, and state replication and failover. That is, XJAF agents are automatically distributed across the cluster nodes in order to share the computational load. In addition, internal state of an agent is automatically copied across other nodes in the cluster. In case of any hardware or software failures, the agent can transparently be restored on one of the remaining nodes.

Again, by analyzing the source code, one may conclude that there is no direct implementation of these features in XJAF. Instead, our framework relies on a modern enterprise application server.

To read more about the features and capabilities of XJAF, please visit the documentation page.

XJAF is released as a free software, and has been tested on Windows and Linux, both 32 and 64-bit versions. Its source code, as well as precompiled binaries can be obtained from the download page.
